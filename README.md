## Desenvolvimento

### Compilar scss em css em tempo real

1. npm install Bootstrap
2. npm install jquery
3. npm install popper.js
4. Instalar rubi e o sass
5. sass --watch node_modules/bootstrap/scss:node_modules/bootstrap/compiler
6. sass --watch node_modules/bootstrap/scss:node_modules/bootstrap/compiler node_modules/font-awesome/scss:node_modules/font-awesome/css view/sass:view/css
